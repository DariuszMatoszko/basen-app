# UI Style Basen1-app

## Styl aplikacji
- Jasny
- Subtelny
- Spokojny
- Bez agresywnych kolorow
- Czytelne wyspy (cards)

## Kolory
- `background`: `#f7f8fa`
- `card`: `#ffffff`
- `border`: `#e6e8ec`
- `text`: `#1f2937`
- `primary-green`: `#22c55e`
- `danger-red`: `#ef4444`
- `soft-gray`: `#9ca3af`

## Elementy UI

### IslandCard
- Biale tlo
- Lekka ramka
- Zaokraglone rogi
- Padding

### CancelButton
- Czerwone kolko
- Bialy X
- Sluzy tylko do anulowania zajec

### AcceptButton
- Zielone kolko
- Bialy check

### GhostButton
- Delikatny szary

## Zasada krytyczna
Podczas zmian UI nalezy korzystac tylko z tych stylow. Nie zmieniac wygladu aplikacji bez powodu.
